#🤖 MedBot AI – Healthcare Awareness Chatbot (Frontend)

MedBot AI is a frontend-based healthcare awareness web platform built using HTML, CSS, and JavaScript. It provides basic disease and symptom information through an interactive chatbot interface. The system uses a structured dataset to simulate responses and demonstrates concepts inspired by ICD-11 and NAMASTE healthcare terminology.

MetaBot AI is a frontend-based healthcare awareness web platform that provides basic information about diseases and symptoms through a chatbot-like interactive interface.

The platform demonstrates how AI-style conversational systems can help users quickly understand health conditions and improve awareness. The system is built completely using HTML, CSS, and JavaScript, with no backend implementation.

🌐 Project Overview

MetaBot AI simulates a health assistant chatbot where users can ask questions about diseases and symptoms.
The chatbot uses a structured dataset of diseases and symptoms implemented in JavaScript to generate responses.

The project also demonstrates integration concepts inspired by:

ICD-11 Disease Classification

NAMASTE Healthcare Terminology

Modern + Traditional Healthcare Knowledge
